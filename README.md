# TPA4Fabric
A simple TPA mod for Fabric.

# Commands
- [x] tpa {to}
- [x] tpaaccept [from]
- [x] tpadeny [from]
- [ ] tpacancel [to]
- [ ] tpaallow {true|false}
- [ ] tpaconfig {key} {value}

> { } is required and [ ] is optional (or in specific cases)
