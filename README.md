# TPA4Fabric
A simple TPA mod for Fabric.

# Commands
- [x] tpa {to}
- [ ] tpahere {target}
- [x] tpaback
- [x] tpaaccept [from]
- [x] tpadeny [from]
- [x] tpacancel [to]
- [x] tpaallow [true|false]
- [ ] tpaconfig {key} {value}

> { } is required and [ ] is optional (or in specific cases)
